Мощное десктопное приложение для автоматической генерации описаний к фотографиям (стикерам) в альбомах VK с использованием современных Vision-моделей через LM Studio.

## ✨ Ключевые возможности

- **🤖 Интеллектуальный анализ**: Распознавание текста и визуальных образов на изображениях.
- **⚡ Двойной режим работы**:
  - **Автоматический**: Быстрая обработка всего альбома без остановки.
  - **Ручной (Интерактивный)**: Просмотр и редактирование каждого результата перед сохранением.
- **🛠️ Продвинутая ручная модерация**:
  - Прямая ссылка на фото в VK для быстрого перехода.
  - Редактирование сгенерированного текста «на лету».
  - Перегенерация описания с вашим комментарием (инструкцией) для нейросети.
  - Пропуск неудачных кадров.
- **🔄 Умные настройки фильтрации**:
  - Выбор направления обхода (сначала новые или старые).
  - Опция повторной обработки фото, у которых уже есть описание.
  - Настраиваемый системный промпт для тонкой калибровки формата вывода.
- **🛡️ Безопасность и стабильность**:
  - Локальный запуск ML-моделей через LMStudio.
  - Автоматическое восстановление соединений и повторы при сбоях API.
  - Отмена последнего действия (Undo) — возвращает старое описание в VK.

## 🚀 Быстрый старт

### 📋 Требования
- **Node.js** >= 24.14
- **Rust** >= 1.70
- **LM Studio** с загруженной Vision-моделью `zai-org/glm-4.6v-flash`
- **VK Access Token** с правами `photos` (получите на [vkhost.github.io](https://vkhost.github.io/))

### 🛠️ Установка и запуск

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/your-username/vk-caption-ai.git
   cd vk-caption-ai
   ```

2. **Установите зависимости**:
   ```bash
   npm install
   ```

3. **Настройте LM Studio**:
   - Запустите локальный сервер (по умолчанию порт `1234`).
   - Загрузите подходящую Vision-модель.

4. **Запустите в режиме разработки**:
   ```bash
   npm run tauri:dev
   ```

### 📦 Сборка
```bash
npm run tauri:build
```
Готовое приложение будет находиться в папке `src-tauri/target/release/bundle`.

## ⚙️ Настройка

В разделе **«Настройки»** внутри приложения заполните:
- **VK Token**: Ваш персональный токен доступа.
- **VK Album ID**: Числовой ID альбома, который хотите обработать.
- **LMStudio Model**: Идентификатор модели `zai-org/glm-4.6v-flash`.
- **Processing Mode**: Выберите «Ручной», если хотите проверять каждое описание перед публикацией.

## 📝 Автоматизация стикеров
По умолчанию приложение настроено на генерацию кратких тегов и поиск текста для стикеров. Системный промпт инструктирует модель возвращать JSON-массив, где первым элементом является текст с картинки, а остальными — поисковые ключевые слова (эмоции, ситуации).
